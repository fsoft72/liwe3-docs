## `warn`


```ts
warn = ( ...args: any )
```


Logs a warning message to the console.
The message is prefixed with a yellow WARN label in magenta background.



**Parameters**

| Name | Type | Description |
| ---- | ---- | ----------- |
| `args` | ...any | The arguments to log to the console. |



**Returns**: `{void}`
